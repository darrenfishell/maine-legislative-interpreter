SUMMARY  16 
Current law requires municipal and city ballot clerks to count all write -in votes in a 17 
municipal or city election regardless of the number of write -in votes cast. This bill 18 
provides that a ballot clerk must count and tabulate the votes cast for a write -in candidate 19 
only if the printed ballot does not include a properly nominated candidate or the number 20 
of write -in votes exceeds the number of votes for a c andidate printed on the ballot.  21
