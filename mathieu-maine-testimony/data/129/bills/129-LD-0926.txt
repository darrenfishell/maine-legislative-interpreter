22SUMMARY
23This bill provides that an assault on a hospital employee while the hospital employee 
24is providing medical care is a Class C crime.  It also provides that an assault on an 
25emergency medical care provider while the emergency medical care provider is providing 
26medical care is a Class C crime.  Current law provides that an assault on an emergency 
27medical care provider while the emergency medical care provider is providing emergency 
28medical care is a Class C crime.  The bill also corrects a cross-reference.
