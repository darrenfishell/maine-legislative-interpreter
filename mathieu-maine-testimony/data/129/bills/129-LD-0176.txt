106.  Release time for state board meeting attendance.  A teacher or administrator 
11serving on the state board must be granted release time for attendance at state board 
12meetings.  The state board is responsible for any cost related to the release time.
13SUMMARY
14This bill eliminates the provision in current law prohibiting teachers and school 
15administrators from serving on the State Board of Education.  It also provides that a 
16teacher or school administrator serving on the state board must be granted release time for 
17attendance at state board meetings.  Any cost related to the release time is the 
18responsibility of the state board.
